# Elastos.ORG.DID.Explorer.Web

## build & run

Install nvm nodejs 
```bash
>$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash
```

Exit the terminal and open it again to activate NVM
```bash
>$ nvm install node
```

Install npm yarn
```bash
>$ sudo apt install npm

>$ npm install -g yarn
```

Install packages
```bash
>$ yarn install
```

Run 
```bash
>$ yarn start
```
